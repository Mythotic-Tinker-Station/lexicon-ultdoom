1. INFORMATION
---------------

The Death Tormention Triligy in a complete replacement for id software's game The Ultimate Doom. It is a
compilation of 4 previously released episodes for Doom, Simply Phobos, and all 3 Death Tormention episodes.

Features:

- 4 Episodes (each of 8 levels and 1 secret level)
- 2 super secret levels exclusive to Zdoom
- Improvements in architecture over the originals.
- Co-op and Deathmatch starts (though this is designed for Single Player)
- All difficulty levels
- Built in MAPINFO
- Built in DEHACKED (Hell Knight Spectre)
- Designed for Zdoom Limit removing source ports

----------------------
2. REQUIREMENTS
----------------------

Although most maps in The Death Tormention trilogy will run under Doom.exe or a limit removing or boom compatible 
sourceport it is recommended that you use Zdoom to play this megawad.

REQUIRMENTS

- The Ultimate Doom Game (Doom.wad)
- Zdoom by Randy Heit (www.zdoom.org) - Enables a few additional animated textures and flats to assure
you get the full experience and play the Death Tormention Trilogy to its full abillities.

Downloading these sourceports guarantees that this wad will run and you'll get many weeks of challenging enjoyment.


---------------
3. INSTALLATION
---------------

Firstly you'll need a copy of the game Ultimate Doom by id software installed on your PC. This is required
since DTPACK will not run standalone.

Unzip DTPACK.WAD into your Doom directory. If you've downloaded Zdoom. Drag DTPACK.WAD into 
the ZDOOM.EXE icon and select Doom.wad.

6. CHANGES TO LEVELS OVER ORIGINAL RELEASES
-----------------------------------------------

All textures from Simply Phobos, Death Tormention 2 and 3 included.

SIMPLY PHOBOS
- Most of SImply Phobos remains as is from the original release.
- Bonus level E2M9 is now the second secret level of Death Tormention.

E1M5 CARMEN'S LAB
- Bridge at start fixed and escape from slime added.

DEATH TORMENTION 1 AND 2
- Architectual tune-ups.
- Improved gameplay challenge in Death Tormention 1.
- Improved gameplay flow in places, less back tracking.
- Sky transfers (Boom / Zdoom)
- Death Tormention 2 custom textures don't overwrite the original Doom textures.

E2M1 LOST AZTEC 
- Mostly unchanged apart from architectual tune-ups.
- Fixed Imps at Blue Armor. Was supposed to be an ambush. Forgot to tag the sector when modified years ago. (v2)

E2M2 MANSION OF SLAYER
- Mostly unchanged apart from architectual tune-ups.

E2M3 CHIRON
- After the blue key was a room with an inescapable pit. You can now escape it and the bridges to
each switch don't take as long.
- Getting the red key no longer requires a backtrack. Red key no longer hidden.
- In the wodden area, getting the yellow key opens up some new ambushes and does not require you
to backtrack through the entire area.

E2M4 THE BUNKER
- Hidden Cyberdemon now contains a BFG9000 which counts as a secret.
- Once you obtain the blue key, bars open up in front of you.

E2M5 DARK OMEN
- Architectual tune-ups, texture alignment. Apart from that, mostly unchanged.

E2M6 DARK HABITAT
- Some improvments in level connectivity.
- Backpack secret near start no longer requires a backtrack to get out.
- Crate room is less mazey with some extra exits added to the area.
- Slime pit around blue key has escapes added.
- Secret exit goes to E2M10 Iron Fist, playable in MAPINFO only ports.

E2M7 THE INMOST CRYPT

- Architectual tune-ups.
- More steps added in the water in the first area of the map to prevent backtracking.

E2M8 MASTERYARD

- Semi hidden switch added to allow exit to be opened. This allows the map to be beaten in source
ports that don't support MAPINFO. Beating the Cyberdemon in non MAPINFO ports will also exit the level.

E2M9 DARK OASIS
- Architectual tune ups.
- Level retextured to more resemble Dr. Sleep's Master Levels.
- A surpise cyberdemon fight added.

E2M10 IRON FIST
- Unchanged.

E3M1

- Improved the flow around the red key area to prevent backtracking.

E3M2 - RETURN TO THE ABYSS
- Second blue key removed and replaced with a switch that requires the blue key to open.
- A few architectua tune-ups.
- Final cave area less Maze-like.

E3M4 - GOING AFTER ASHLEY!
- Unchanged apart from texture alignments and a slight architectual overhaul.

E3M5 PAIN TEMPLE
- Improvments to the level's connectivity.
- Blue key does not require backtracking all the way back to the staircases and the windows
now lower.
- Red key required unessecary backtracking so it was removed and the small central metal
room was made more open allowing access from both sides without the need for unessecary backtracking.
Also another yellow key door added that connects to the Twilight Zone room with blue key.

E3M8 THE LAST HUNT
- Teleports added into pits allowing you to escape.
- If playing in a sourceport that does not support MAPINFO. Beating the Spider Mastermind will end
the level instead of lowering the exit.
- Hidden switch added to allow exit to be opened allowing the map to be completed in Nomonsters mode
in non MAPINFO ports.

DEATH TORMENTION 3
- Most levels remain as is with a few minor cosmetic changes.
- A few changes in the music selection.
- E4M7 secret pool room contains a new cartoon female pool character to match the others replacing a
poor qualiy greyscale newspaper photo.
- E3M9 Unbleed is now E4M10.
